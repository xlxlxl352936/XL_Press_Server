# XL Press Server

XL Press（Android）と連携するPCサーバーツールです。

## 機能
- PCの動画をスマホでストリーミング再生
- PCの動画をスマホのFixフォルダにダウンロード
- スマホのFixフォルダからPCへ動画をアップロード
- QRコードで簡単接続

## 必要環境
- Windows 10 / 11
- 同一Wi-Fiネットワーク

## 使い方
1. `XL_Press_Server.exe` をダウンロードして起動
2. 初回起動時にパスワードを設定
3. `UNLOCK` でログイン
4. `START SERVER` を押してサーバーを起動
5. `Show QR` でQRコードを表示
6. スマホのXL PressアプリのPCタブでQRコードをスキャンして接続

## 注意事項
- スマホとPCが同じWi-Fiに接続されている必要があります
- ファイアウォールでポート8000の許可が必要です
- 動画はPC上の暗号化フォルダに安全に保存されます
